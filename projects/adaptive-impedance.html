Research Case Study

Adaptive Compliance
Control Framework.

Developing a closed-loop attention mechanism for variable impedance control in multi-arm manipulator systems.

Control Theory Fig 1.0: Compliance Controller Pipeline

Implementation of the torque-based control loop featuring gravity and friction compensation, converted to current commands for the Kinova Gen3 Lite actuators.

Software Stack Fig 2.0: ROS 2 Humble & Docker Architecture

Docker Containerization

Environment isolation using Docker Host to manage Python-based Kortex clients and ROS 2 middleware simultaneously.

ROS 2 Humble Nodes

Distributed nodes for compliance control, logging, and Gazebo simulation integration with custom message interfaces.

Hardware Interface

Communication via Ethernet (TCP/UDP) for the Kinova arms and USB CAN for the mobile ceiling rail system.

Execution & Validation

Validation on Kinova Gen3 Lite arms mounted on a multi-axis rail system.

Mathematical Formulation

$$ M(q)\ddot{x} + D(q, \alpha)\dot{e} + K(q, \alpha)e = F_{ext} $$

The objective is to minimize contact force $F_{ext}$ by optimizing the attention weights $\alpha$, which modulate the mass, damping, and stiffness properties of the manipulator.

Implementation Details

01

Real-Time Dynamics: Implemented using the Pinocchio library for high-speed computation of the RNEA (Recursive Newton-Euler Algorithm) within a 1kHz control loop.

02

Attention Strategy: Using Multi-Head Attention to process past force trajectories and joint velocities to predict task-dependent compliance requirements.
